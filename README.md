# TimeTravel Agency - Aurélie Runser - M2 Dev Ynov Strasbourg

Landing Page pour une agence de voyage temporel fictive, créée avec IA générative.

## 🛠️ Stack Technique
- HTML 5 / CSS 3 / JavaScript
- Node.js / Mistral AI API
- Hébergement : Render

## ✨ Features
- Landing page avec
    - Galerie de 3 destinations temporelles
    - Formulaire de reservation
    - FAQ
    - Chatbot IA conversationnel

## 🤖 IA Utilisées
- Code : GitHub Copilot & ChatGPT
- Chatbot : Mistral Small via API
- Visuels : exactly.ia + mistral.ia + leonardo.ia + Runway (Projet 1)

## Installation & Développement
- Assurer d'avoir un environnement *Node.js*
- Dupliquer ou télécharger le projet en local
- Dupliquer le fichier **.env.example**, renommer la copie **.env** et renseigner une clé Api Mistral
- Dans le terminal ouvert dans le projet, tapper `npm init -y` pour installer les dépendances
- Vous pouvez ensuite y tapper `node server.js` pour lancer l'application en local
